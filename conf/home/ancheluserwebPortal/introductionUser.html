Anchel Web Portal

Launch any of the supported web browsers on your system and enter Anchel Link in the address bar. The Anchel landing page is then presented.

  • Click on the icon in Services to launch the ‘End User’ landing page.
  • The user will then be presented with a login dialog. Enter the registered user name and password in the dialog.
  • After successful authentication, depending on the roles (refer to the admin guide in for roles in Services) available to the user, the home page will be
  • The minimal modules and information available in the home page include:
    • Copyright
    • Version of the application
    • Available storage space
    • User information
    • Preferences
    • Host your Files module
    • Activity logs
    • Upload and Share
    • Upload to a workgroup (Role based access)
