Host Your Files

Click on HOST YOUR FILES and expand 'My Space' and Select 'My Files'.


Upload a File

Click on     and then click on     File Upload.


Select file from the file system.


Status

Once file is selected the upload will be initiated and the upload status can be seen by clicking the image link as view.


Perform

User can perform operations such as Share, Play, Pause, Cancel and the appropriate action will be taken as applicable to the file with the default application.
