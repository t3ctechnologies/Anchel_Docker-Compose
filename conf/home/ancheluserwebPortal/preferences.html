Contact List

Click on preferences it should display contact list and guest as given below. Click on contact list or by default it should display contact list page.

Add Contact List

Click on upload button .It Should create contact list as given.

Delete Contact List

Select contact list and Click on delete button in header menu it should display alert popup. Click on YES DELETE THEM.


After delete contact list

Add Contact

Click on upload contact List It will create contact list


Click on add contact button in item menu.


Once add Contact Button Click it should display add contact popup.


Search contact by email and click on email. It should add contact as given.

Delete Contact

Select single contact and click on delete button in Heder menu.


It should display alert Popup Now click on YES DELETE THEM.


Once click on delete contact it will delete contact.


Search By Or Filter By

User can use Filter by option to check my list and Order list. User can Search contact by Search by Name tab bar.


SortBy

Click on sort by button.User can see all options (List name, private/public, modification date and create date) of sort type.


More Action

User can click on more Action button in item menu to perform (information , show contact, add contact,delete and rename).
