We are providing the secured connection using SSL , information you send or get through this site is private. Distribution of workloads across multiple computing resources by using amazon EC2 load balancer.

Application consist of mainly three web server. The first server deals with authentication , authorization using LDAP, SMTP configuration, scanning of file using clam antivirus and encrypting/decrypting of file using Base64 mechanism. The second and third server will concentrate on massive file transfer with executing some pre and post step actions, finally file will get store into amazon S3 bucket.
