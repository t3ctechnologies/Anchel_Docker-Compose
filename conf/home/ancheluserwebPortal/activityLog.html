Activity Log page

Click on the 'Activity Log' in the navigation panel to list a history of actions performed by the user.


Calendar

The activity logs can be further filter down to activity within a range of dates using the calendar icon provided in the toolbar.


Sort By Order

User can sort all activity (Actor, Action, Resource type, Resource name, Recipient and Date) by sort by or order by button as view.


Information

User can see information of one action by click on information button as view.


Pagination

Number of records displayed in a single page can be customized with the display setting available at the bottom of the page.
