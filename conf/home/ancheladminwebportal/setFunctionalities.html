Functionalities

Click on functionalities it will Display all kind of functionalities.


Anonymous Url

Click on Anonymous URL it will open Anonymous URL permission page.Enable functionality can allow user to Send data file in any email.


Click on Advance it will open advance Permission popup of anonymous url. Click on all option (Mandatory, allowed, forbidden).

  • Mandatory: User must have right to send file on any email.
  • Allowed: User allowed to send file on any email.
  • Forbidden: User do not have right to send file on any email.
